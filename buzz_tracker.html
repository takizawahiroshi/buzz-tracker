直接開けません
ブラウザのセキュリティにより、file:// からAPIへの接続はブロックされます。
以下の手順で起動してください。
1
server.py と buzz_tracker.html を同じフォルダに置く
両ファイルを同じディレクトリに保存してください。
例: C:\Users\name\buzz-tracker\
2
コマンドプロンプトでサーバーを起動
Windows — そのフォルダで右クリック →「ターミナルで開く」
または cd でフォルダに移動してから:
python server.py sk-ant-api03-XXXXXXXX クリックでコピー
※ sk-ant-api03-XXXXXXXX の部分は実際のAPIキーに置き換えてください
3
ブラウザで localhost:8000 を開く
サーバーが起動したら、このURLをブラウザで開いてください:
http://localhost:8000 クリックでコピー
簡単起動 (Windows): ダウンロードした start.bat をダブルクリック
→ APIキーを入力するだけで自動起動＆ブラウザが開きます
🔑 APIキーの取得先 console.anthropic.com/settings/keys でAPIキーを作成してください
